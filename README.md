# Expense Tracker (DevOps Enabled)

This is an Expense Tracker application built with React (frontend) and Node.js/Express (backend).  
It allows users to manage expenses efficiently with authentication and filtering features.

<img src="overview_img.png">

---

## 🚀 Features
- Add, edit, delete expenses
- Filter by month and year
- User authentication (JWT)
- Responsive mobile-first UI

---

## ⚙️ Tech Stack
- **Frontend:** React, Redux, TailwindCSS, react-toastify  
- **Backend:** Node.js, Express, MongoDB  
- **Auth:** JWT  
- **DevOps:** GitHub Actions, Docker (planned), CI/CD pipeline
